# Community Map Tool

Small static browser tool for creating simple printable community maps from OpenStreetMap. Developed by SCDC (www.scdc.org.uk) and available at maps.scdc.org.uk.

## How to use

- Search for a place name.
- Pan and zoom the map, then lock an export frame to A1-A5 portrait or landscape ratios.
- Fine-tune the frame with small `Frame In` and `Frame Out` steps.
- Download PDF, PNG, or GeoJSON exports.
